# Shiba Engine 2D!
2D game engine, using SDL2. 
Made purely for fun and as a learning experience.

---
## To compile and run
Make sure you have SDL2 installed.  
```sudo pacman -S sdl2 sdl2_image sdl2_ttf sdl2_mixer sdl2_net```  

```cd Shiba-Engine-2D```  
```make```  
```make run```  
```make debug```  
```make clean```
